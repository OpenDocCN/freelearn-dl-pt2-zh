# 第三部分：部署与维护

在部署深度学习项目过程中经常会面临许多复杂挑战。在许多情况下，部署环境与开发环境不同，这种差异可能会引入各种限制。在本部分中，我们介绍工程师经常遇到的常见问题，并针对每个挑战分享有效的解决方案。在最后一章中，我们描述了深度学习项目的最后阶段，包括评估项目并讨论未来项目的潜在改进。

本部分包括以下章节:

+   *第八章*, *简化深度学习模型部署*

+   *第九章*, *扩展深度学习管道*

+   *第十章*, *提升推理效率*

+   *第十一章*, *移动设备上的深度学习*

+   *第十二章*, *监控生产中的深度学习端点*

+   *第十三章*, *审查完成的深度学习项目*
